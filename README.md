# pyex
Digital currency exchange API For Python
Python的数字货币交易API

pyex目标：pyex是统一的标准化的数字货币交易API，支持多个平台。

此项目属于quantmew自动化交易平台项目数字货币组件的一部分。

```
cd pyex

```
